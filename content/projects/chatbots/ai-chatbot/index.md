## Overview
Enterprise-grade AI chatbot built with LangGraph and ReAct framework. Features intelligent tool orchestration and multi-agent collaboration.

## Key Features
- **RAG Integration**: Context-aware responses using vector databases
- **Multi-LLM Support**: OpenAI, Anthropic, Azure OpenAI
- **Specialized Agents**: Web search, calculator, email automation
- **Vector Databases**: ChromaDB, FAISS, Pinecone support

## Capabilities
- Intelligent tool orchestration
- Configuration-driven architecture
- Evaluation framework
- Multi-cloud deployment ready

## Tech Stack
Python • LangChain • LangGraph • Streamlit • ChromaDB • Docker • FastAPI
